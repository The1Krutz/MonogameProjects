# Steel Tower

## Introduction

I tried a mobile game once where there was some kind of tower in the middle and enemies kept coming at it. The concept was neat, but you could see where they were leaving room for all the mobile microtransaction needles to get you. I went two rounds and was already begging for a way to speed up progression: that's how they get you!

So I'm going to recreate it for fun, except without all the boring bullshit that tricks you into spending real money in the mobile microtransaction hellscape!
